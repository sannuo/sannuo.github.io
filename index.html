Title

Web-Based Data Structure Visualization

Summary

One challenge remains in teaching and learning data structure is to facilitate students’ understanding of the dynamic presentation from the collection of data elements in memory. Visualizing how each element points to the next, and how a collection of nodes together represents a sequence with graphs or linked-node diagram can help us understand not only the abstract data structure but also the dynamics of the data elements in the program execution. While printing (to a console) is a very common technique for understanding and debugging data structures, it is very limited for complex structure depiction.

This research will focus on visual and interactive methods to facilitate the communication of data structure and associated data operations by dynamic graphing corresponding to the programming process. The main features will include: 1) Dynamic depiction. By using the inner execution trace, the program will be able to monitor the run-time data structure change and render the corresponding presentation in real-time. The resulting serial of key frames featuring major data structure transformation will be available for review and analysis. 2) Profile visualization. With profiling libraries in Python, the research will cover a serial of profiler visualization charts to trace an instance time complexity and space complexity. 3) Animation. A serial of animation will be designed in an electrical lecture mode to illustrate the obscure concepts in data structure and algorithms. All these aims to provide a comprehensive exploration and fill the gap between the abstract data representation and a dynamic learning-by-coding process.
